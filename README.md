## How to download the project:

```bash
# Clone the repository
$ git clone https://github.com/rodrigoaraujo7/tech-list.git

# Enter directory
$ cd tech-list
```

## Usage

1. Run `yarn install`.<br />
2. Run `yarn dev` and access `http://localhost:3000`.<br />
